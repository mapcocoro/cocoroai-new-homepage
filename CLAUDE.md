# ココロAI コーポレートサイト

ココロAI合同会社の公式Webサイト

## 技術スタック

- HTML5 + Tailwind CSS (CDN)
- JavaScript (Vanilla)
- Google Fonts (Noto Sans JP)
- ホスティング: GitHub Pages

## サイト構成

| ファイル | 内容 |
|---------|------|
| `index.html` | トップページ（事業内容・プロダクト紹介・ニュース・会社概要） |
| `pricing.html` | 料金プランページ（4カテゴリのタブ切り替え式） |
| `solutions.html` | お悩み別ソリューションページ |
| `faq.html` | よくある質問ページ |
| `style.css` | 共通スタイル |
| `samples/` | 制作サンプル（美容サロンテンプレート等） |
| `images/` | 画像アセット |
| `videos/` | 動画アセット（デモ動画等） |

## 開発時の注意

- GitHub Pagesでホスティングしているため、リポジトリはPUBLICのままにすること
- PRIVATEにするとサイトが停止する
- 全ページ共通でヘッダー・フッターを手動で同期する必要あり
